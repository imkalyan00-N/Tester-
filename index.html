K-Means Clustering

PART 1: Introduction

The k-means algorithm can be described in the following steps:

K-Means clustering ante data ni k groups (clusters) lo divide cheyyadam. Idi unsupervised learning method (labels undavu). Goal → similar data ni okate group lo petadam.
PART 2: Step 1

Select k initial cluster centres from the given n data points. The remaining (n-k) data points are then assigned to one of the k clusters based on their proximity to the closest cluster centre.

Step 1: - k centres random ga select chestaru - migatha points ni nearest centre ki assign chestaru Example: 3 clusters unte → 3 centers select chestaru
PART 3: Step 2

Compute the new cluster centres based on the current assignment of data points. This is done by calculating the mean or centroid of all the data points belonging to each cluster.

Step 2: - prathi cluster lo unna points average (mean) calculate chestaru - new centre (centroid) create avtundi Centroid = cluster middle point
PART 4: Step 3

Assign each of the n data points to the cluster centre that is closest in proximity to it.

Step 3: - malli anni points ni nearest new centre ki assign chestaru - clusters update avtayi
PART 5: Step 4

Check if there have been any changes in the assignment of data points to clusters. If no changes have occurred, terminate the algorithm; otherwise, go back to Step 2 and repeat the process.

Step 4: - clusters change ayaya check chestaru If change lekapothe: → STOP If change unte: → malli Step 2 ki vellali Idi loop laga continue avtundi
PART 6: Final Objective

By iteratively updating the cluster centres and reassigning data points until convergence, the k-means algorithm aims to find a partition that minimizes the overall within-cluster variance or distance.

Final goal: - cluster lo unna points madhya distance minimum undali Simple ga: 👉 same type data okate cluster lo undali 👉 different data vere cluster lo undali Convergence ante: - clusters stable ayipovadam (inka change kakapovadam)
